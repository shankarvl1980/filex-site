Privacy Policy

Last updated: January 2026

FileX Manager is an Android file management application that allows users to manage files stored on their device and optionally connect to third-party cloud services such as Google Drive.

Google User Data Access

When you connect your Google Drive account, FileX Manager accesses Google user data strictly to provide user-initiated file management features such as browsing, uploading, downloading, copying, moving, renaming, and deleting files and folders.

FileX Manager does not access Google Drive data unless the user explicitly connects their account and initiates actions within the app.

Data Protection and Security Measures

FileX Manager implements the following safeguards to protect Google user data:

  • OAuth 2.0 is used for secure authentication with Google Drive.
  • Access tokens are stored only on the user’s device and are never transmitted to external servers.
  • All communication with Google APIs occurs over encrypted HTTPS connections.
  • No Google user data is stored on developer-controlled servers.
  • The app follows the principle of least privilege and accesses data only as required to perform user-requested actions.

Data Storage

FileX Manager may store the following data locally on the user’s device:

  • OAuth access tokens required to maintain the Google Drive session
  • Temporary file metadata and cache to improve performance

This data remains solely on the user’s device and is not shared with the developer or third parties.

Data Retention and Deletion

FileX Manager retains Google user data only for as long as necessary to provide its functionality:

  • Access tokens and cached metadata are retained only while the Google Drive account remains connected.
  • When the user disconnects their Google Drive account, all associated tokens and cached data are deleted immediately from the device.
  • Users may also delete all locally stored data at any time by clearing the app’s data or uninstalling the app.

FileX Manager does not retain Google user data after account disconnection.

Data Sharing

FileX Manager does not sell, rent, or share Google user data with third parties. Data is disclosed only if required to comply with applicable laws or legal processes.

User Control

Users have full control over their data and may revoke Google Drive access at any time through:

  • The FileX Manager app (disconnecting the account)
  • The Google Account security settings
