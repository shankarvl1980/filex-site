FileX Manager - Android File Manager

FileX Manager is an Android file manager that supports local storage, USB, network storage (SMB, WebDAV, FTP, SFTP), and cloud storage such as Google Drive.

FileX Manager uses Google Drive access only to provide file browsing, upload, download, and management features initiated by the user.
